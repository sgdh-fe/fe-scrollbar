# 模拟滚动条

<br />

浏览器模拟滚动条 `兼容ie7+` 

[预览地址](https://sgdh-fe.github.io/fe-scrollbar/)

<br />


![demo](demo.gif)

<br />


## 使用说明

### 开发环境运行

```sh
yarn 或 npm install

```

### 开发预览

```sh
yarn start 或 npm run start
```

<br />


---

### 使用

```js
//导入
src/fe.scrollbar.js

//初始化
initScrollBar("feScrollbar")

```
